# taiwan-night-market-calorie-database

台灣夜市小吃熱量資料庫。收錄 35+ 種常見夜市食物的熱量、蛋白質、脂肪、碳水化合物，
以及減重友善選擇指南，數據來源為衛福部食藥署 TFND 及同行評審研究。

## 為什麼建立這個資料庫

夜市是台灣 90% 以上民眾的日常外食場景，卻缺乏系統化的熱量資料庫。
現有資料散落於各健康部落格，格式不一且缺乏蛋白質、脂肪等完整巨量營養素數據。
本資料庫以衛福部食藥署 TFND 為主要來源，提供結構化 JSON 格式，
供減重計畫、飲食追蹤應用程式和研究者使用。

## 資料說明

| 檔案 | 說明 |
|------|------|
| `data/street-food-calories.json` | 35+ 夜市小吃熱量、蛋白質、脂肪、碳水（每份標準份量） |
| `data/smart-choices.json` | 減重友善選擇（熱量 < 400 kcal 且蛋白質 > 8g）及避雷選項 |
| `data/portion-guide.json` | 常見份量估算（目視法）及點餐調整建議 |

## 熱量分級（每份）

| 等級 | 熱量 | 說明 |
|------|------|------|
| 輕量 | < 250 kcal | 減重期可選 |
| 適中 | 250–400 kcal | 控制份量下可接受 |
| 高熱量 | 400–600 kcal | 偶爾享用，注意當日總量 |
| 極高熱量 | > 600 kcal | 一份即佔每日熱量 30%+ |

## 資料來源

- 衛福部食藥署台灣食品成分資料庫（TFND）
- USDA FoodData Central（部分原料參考）
- 衛福部國民健康署「外食族健康飲食指南」

## 相關工具

- [食物熱量查詢](https://metabolab.tw/calculators/food)：查詢台灣常見食物詳細營養數據
